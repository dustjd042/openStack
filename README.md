




# [OpenStack을 이용한 클라우드 시스템 구축] (4) 환경구성 1
### OpenStack 구축 방법
1. Installation: 가장 기본적인 수동적인 구축 방법 (해당 수업 실습 방법)
2. DevStack: 개발 환경 구성 신속하게 1회성 구축을 위한 방법
3. Deployment: 운영 환경 구성을 위한 구축 방법 (자동 배포 도구를 활요)
